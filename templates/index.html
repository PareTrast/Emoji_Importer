How to Import

üçé iOS (Requires Mac)

  1. Download the .plist file to your Mac.
  2. Open System Settings → Keyboard → Text Replacements.
  3. Drag and drop the downloaded file into the list.
  4. The shortcuts will sync to your iPhone via iCloud.

ü§ñ Android

  1. Download the .csv file to your phone.
  2. Go to your Keyboard Settings (e.g., Gboard or Samsung Keyboard).
  3. Look for Dictionary or Text Shortcuts.
  4. Select Import and choose the CSV file.
  5. Note: Some keyboards may require a third-party "Personal Dictionary" app to import CSVs.
